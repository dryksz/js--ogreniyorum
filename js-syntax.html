JAVASCRİPT SYNTAX (YAZILIM) YAPISI

* JavaScript’in yazılım kurallarının (Syntax) çoğu JAVA programlamaya benzer ancak JavaScript aynı zamanda Perl ve Python’dan da etkilenir.

* JavaScript dilinde alfanumeric değişkenler ve düzenli ifadeler (RegExp) gereği olmadıkça, her türlü boşluk, satır aralığı ve tab karakterlerini göz ardı eder.

* JavaScript’te bütün değişkenler harf, alt çizgi veya alt bağlaç ( _ ) veya dolar ( $ ) işareti ile başlar.

* Rakamlar ve diğer karakterler bu üç karakterden sonra kullanılır.

* JavaScript büyük küçük harf duyarlıdır (Case Sensitive).

JavaScript’te Yorum Satırı Ekleme

* Javascript’te yorum satırı ekme tıpkı C++ ve diğer dillerde olduğu gibidir.

* Tek satır açıklama yazmak için satırın başına // metin yazmanı yeterliyken birden fazla satırlı yorum veya açıklama yazmak için metninizi /* metin */ şeklinde yazmalısınız.

* Yorumlar ve açıklama satırları yorumlayıcılar (interpreter) tarafından dikkate alınmaz. Dolayısıyla sisteminize de herhangi bir işlem yükü getirmeyecektir.

JavaScript Tanımlayıcıları (Identifiers)

*Tanımlayıcılar, JavaScript değişkenlerini, fonksiyonlarını, nesnelerini ve tüm diğer ögeleri adlandırmak üzere kullanılabilen alfasayısal karakterler topluluklarıdır.

*Görevleri, belirli bellek bloklarının, program boyunca belirli bir tanımla çağrılmasını sağlamaktır. Yani, tanımlayıcılar birer isimdir
