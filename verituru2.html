String (Metin)

Bir string harf, sayı gibi karakter bileşenlerinden oluşan türdür. Örneğin "Ali Veli", "Numara13", "abc" birer string türdür.

JavaScript programlama dilinde string veri türü tırnak içinde yazılır. Bu tırnak çift veya tek olabilir:

------------------------------------------------------------------

var kisi = "Derya";

var kisi = 'Derya';

------------------------------------------------------------------

Number (Sayı)

JavaScript programlama dilinde sadece tek tür sayı vardır. Bir değişkene sayılsal değer atanıyorsa bu değer tırnak içinde yazılmaz. Sayılarda ondalık bölüm nokta ile yazılır:

------------------------------------------------------------------

var sayi1 = 10;

var sayi2 = 9.99;

Boolean (Doğru/Yanlış)

Boolean veri türü sadece iki değer alır: true (doğru) ve false (yanlış)

------------------------------------------------------------------

var ogrenciMi = true;

var aktifMi = false;

Boolean veri türü genelde koşul ifadelerinde kullanılır. Koşul ifadeleri ile ilgili ilerleyen derslerde daha fazla bilgi verilecektir.
